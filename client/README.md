# File Sharing App - Client

Frontend for a full stack file sharing application. Provides a clean UI for uploading, sharing, and managing files.

## Features

- Drag and drop file uploads
- Real time upload progress
- Instant shareable links
- QR code generation for file links
- Anonymous and authenticated user support
- Dashboard for managing uploaded files
- Copy, delete, and track downloads

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- Supabase (Auth integration)

## Getting Started

### Prerequisites

- Node.js 18+
- Running backend server
